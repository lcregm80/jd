## 产品线：轻量云主机 vs 云主机 CVM，傻傻分不清楚？

京东云的云服务器主要分两条线：

**轻量云主机**，套餐式销售，配置固定，月流量有上限，但价格非常给力，适合个人建站、小型项目、开发测试、游戏联机。

**云主机 CVM**，更像是传统云服务器的打法，不限流量，配置更灵活，价格偏高，适合企业级应用、高并发业务。

两者各有适用场景，不是说 CVM 更好，而是看你业务用途。比如你就是想搭个个人博客，拿 CVM 当大炮打蚊子，性价比就不高了。

## 实际体验怎么样？

从已有的第三方测评来看，京东云轻量云主机的计算性能中规中矩，采用英特尔至强处理器，单核多核跑分表现尚可，网络延迟在国内用户体验上属于正常水准。存储方面用的是 SSD 云硬盘，I/O 性能比老款 HDD 好不少。

整体来说，在国内云厂商里，京东云的性价比确实属于偏高水平——尤其是轻量云主机，同配置比很多竞品要便宜一截，适合资源有限但需要稳定云环境的用户。
